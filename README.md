# FranTool
简单封装的ApkTool，jadx，dex-tools 以达到减少命令行调用

# 使用要求

1、Java 8  

2、设置JAVA_HOME

# 在window下使用步骤如下：

1. 下载[tool.zip](https://github.com/qq821978827/FranTool/releases/download/tool/tool.zip)
2. 解压到自己喜欢的位置
3. 将解压的位置添加到环境变量path
4. 双击需要的.reg
5. 已经可以正常使用了
